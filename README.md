## Description

A basic API build-in Nest and Apache Kafka to manage notifications

## Installation

```bash
$ yarn
```

or

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## ENV

Rename .env-example to .env with the correct values

You can use https://console.upstash.com/kafka to create a Kafka cluster

## Running Apache Kafka producer

```bash
$ node scripts/producer-notifications.send-notification.mjs
```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```
